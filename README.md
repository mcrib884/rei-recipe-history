# REI Recipe History

A Minecraft mod that adds recipe history to Roughly Enough Items.


## Requirements

- Minecraft 1.19.2
- Fabric Loader 0.14+ or Forge 43+
- Roughly Enough Items 9.x

## Installation

1. Download the latest release for your mod loader
2. Place the jar in your mods folder
3. Launch the game
